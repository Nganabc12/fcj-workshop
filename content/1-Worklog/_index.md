# WORKLOG

This worklog summarizes my **12-week internship journey** at **AWS First Cloud Journey**, from **April 17, 2026 to July 20, 2026**.

Throughout the internship, I gradually built my cloud computing knowledge while participating in the development of the **Pet Resort & Care System**. My responsibilities included setting up the development environment, learning core AWS services, designing cloud architecture, developing and deploying applications, integrating system components, testing, troubleshooting, and preparing the final project documentation.

**Week 1:** [Development environment setup, Git, GitHub, Hugo, Visual Studio Code, AWS account activation and Markdown documentation](1.1-week1/)

**Week 2:** [AWS Global Infrastructure, IAM, MFA, AWS CLI and basic Amazon EC2 practice](1.2-week2/)

**Week 3:** [Amazon VPC networking, Internet Gateway, Route Tables, Public & Private Subnets and network architecture design](1.3-week3/)

**Week 4:** [Amazon EC2 deployment, Linux environment configuration, SSH access and Security Groups](1.4-week4/)

**Week 5:** [Amazon S3, Static Website Hosting, Object Versioning, Lifecycle Rules and storage management](1.5-week5/)

**Week 6:** [Amazon RDS MySQL, database design, Spring Boot database connectivity, backup and snapshot management](1.6-week6/)

**Week 7:** [Project requirement analysis, AWS architecture design, system flow diagrams and deployment planning](1.7-week7/)

**Week 8:** [Spring Boot project structure, REST API development, Maven build process and Postman testing](1.8-week8/)

**Week 9:** [Backend deployment on Amazon EC2, Application Load Balancer, Target Groups, Health Checks and project integration](1.9-week9/)

**Week 10:** [Frontend and backend integration, Amazon CloudFront, debugging, API testing and log analysis](1.10-week10/)

**Week 11:** [Amazon CloudWatch monitoring, AWS Budgets, system testing, performance optimization and bug fixing](1.11-week11/)

**Week 12:** [Final deployment verification, project documentation, internship report preparation and final presentation](1.12-week12/)
