This section summarizes the activities and progress completed during my **12-week internship** at **AWS First Cloud Journey**, from **April 17, 2026 to July 30, 2026**.

Throughout the internship, I gradually built my cloud computing knowledge while developing the **Pet Resort & Care System**. In addition to learning core AWS services, I participated in requirements analysis, system architecture design, application development, AWS deployment, system testing, performance optimization, and technical documentation.

**Week 1:** [Development environment setup, Git, GitHub, Hugo, Visual Studio Code, AWS account creation, and Markdown basics](1.1-week1/)

**Week 2:** [Learning AWS Global Infrastructure, IAM, MFA, AWS CLI, and basic Amazon EC2 practice](1.2-week2/)

**Week 3:** [Learning Amazon VPC, Internet Gateway, Route Table, Public & Private Subnets, and network architecture design](1.3-week3/)

**Week 4:** [Working with Amazon EC2, Linux environment configuration, SSH connection, and Security Group management](1.4-week4/)

**Week 5:** [Learning Amazon S3, Static Website Hosting, Object Versioning, Lifecycle Rules, and storage management](1.5-week5/)

**Week 6:** [Learning Amazon RDS MySQL, database design, Spring Boot database integration, backup, and snapshots](1.6-week6/)

**Week 7:** [Project requirement analysis, AWS architecture design, system diagram creation, and deployment planning](1.7-week7/)

**Week 8:** [Spring Boot application development, REST API implementation, Maven dependency management, and API testing with Postman](1.8-week8/)

**Week 9:** [Application deployment, environment configuration, security enhancement, and CloudWatch monitoring](1.9-week9/)

**Week 10:** [Architecture refinement, system integration, configuration optimization, and issue resolution](1.10-week10/)

**Week 11:** [System testing, AWS resource monitoring, performance optimization, cost management, and troubleshooting](1.11-week11/)

**Week 12:** [Project finalization, technical documentation, workshop completion, internship report preparation, and final system verification](1.12-week12/)
