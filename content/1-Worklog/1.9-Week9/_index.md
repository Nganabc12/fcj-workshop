### Weekly Summary

During Week 9, I focused on deploying the **Pet Resort & Care System** to the AWS Cloud environment and ensuring that all application components worked together correctly after deployment. I reviewed the deployment architecture, configured the application environment, verified the database connection, and checked network communication between services.

To improve system reliability, I reviewed the security configuration by checking Security Groups, IAM permissions, and network settings. I also configured **Amazon CloudWatch** to monitor application logs, resource utilization, and system performance, allowing deployment issues to be detected and resolved more efficiently.

Throughout the deployment process, I encountered several configuration and connectivity issues, including incorrect environment variables and service communication problems. By troubleshooting these issues, adjusting the application configuration, and redeploying the system, I successfully achieved a stable deployment.

At the end of the week, I optimized the deployment configuration, updated the project documentation, and reviewed the entire deployment workflow. This helped me gain practical experience with AWS deployment, cloud security, monitoring, troubleshooting, and operational best practices for cloud-based applications.
