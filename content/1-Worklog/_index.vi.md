Phần này tóm tắt quá trình **12 tuần thực tập** của tôi tại **AWS First Cloud Journey**, từ **17/04/2026 đến 30/07/2026**.

Trong suốt thời gian thực tập, tôi từng bước xây dựng kiến thức về điện toán đám mây thông qua việc học tập và phát triển dự án **Pet Resort & Care System**. Bên cạnh việc tìm hiểu các dịch vụ cốt lõi của AWS, tôi còn tham gia phân tích yêu cầu, thiết kế kiến trúc hệ thống, phát triển ứng dụng, triển khai lên AWS, kiểm thử, tối ưu hệ thống và hoàn thiện tài liệu dự án.

**Tuần 1:** [Thiết lập môi trường phát triển, Git, GitHub, Hugo, Visual Studio Code, tạo tài khoản AWS và làm quen với Markdown](1.1-week1/)

**Tuần 2:** [Tìm hiểu AWS Global Infrastructure, IAM, MFA, AWS CLI và thực hành cơ bản với Amazon EC2](1.2-week2/)

**Tuần 3:** [Tìm hiểu Amazon VPC, Internet Gateway, Route Table, Public & Private Subnet và thiết kế kiến trúc mạng](1.3-week3/)

**Tuần 4:** [Làm việc với Amazon EC2, cấu hình môi trường Linux, kết nối SSH và quản lý Security Group](1.4-week4/)

**Tuần 5:** [Tìm hiểu Amazon S3, Static Website Hosting, Object Versioning, Lifecycle Rule và quản lý lưu trữ dữ liệu](1.5-week5/)

**Tuần 6:** [Tìm hiểu Amazon RDS MySQL, thiết kế cơ sở dữ liệu, kết nối Spring Boot với cơ sở dữ liệu, sao lưu và Snapshot](1.6-week6/)

**Tuần 7:** [Phân tích yêu cầu dự án, thiết kế kiến trúc AWS, xây dựng sơ đồ hệ thống và lập kế hoạch triển khai](1.7-week7/)

**Tuần 8:** [Phát triển dự án Spring Boot, xây dựng REST API, quản lý Maven và kiểm thử bằng Postman](1.8-week8/)

**Tuần 9:** [Hoàn thiện các chức năng của hệ thống, triển khai ứng dụng lên AWS và tối ưu cấu hình môi trường](1.9-week9/)

**Tuần 10:** [Hoàn thiện kiến trúc triển khai, tích hợp các thành phần của hệ thống và xử lý các lỗi phát sinh](1.10-week10/)

**Tuần 11:** [Kiểm thử hệ thống, giám sát tài nguyên AWS, tối ưu hiệu năng, quản lý chi phí và xử lý sự cố](1.11-week11/)

**Tuần 12:** [Hoàn thiện dự án, xây dựng tài liệu kỹ thuật, hoàn thành Workshop, chuẩn bị báo cáo thực tập và kiểm tra tổng thể hệ thống](1.12-week12/)
